8 Reverse Correctness Presentation   Homework_5_1

In lecture and in the homework, we have stressed that assignment command calculations should ‚Äòbegin at the bottom‚Äô and work up; as such, it is more ‚Äònatural‚Äô to begin our calculation with the ‚Äòbottom part‚Äô at the top and work our way down to the other part.

To do so, we use the ‚Äòflipped around‚Äô assignment connective: 

    Axiom “Definition of ⁅_⁆⇐”:   Q ⁅ C ⁆⇐ P  ≡  P ⇒⁅ C ⁆ Q

This is similar to the arithmetic ordering: 

    y ≥ x  ≡  x ≤ y

It is the same thing, but the order has been flipped to make writing our calculations the same way we actually think of them: With the post-condition first, then eventually obtaining the pre-condition.

Here is a demonstration… 

     Lemma “Demo”: x + y = 13    ⇒⁅ x := x + 7 ⁆   x + y = 20
     Proof:
         x + y = 20
       ⁅ x := x + 7 ⁆⇐ ⟨ “Assignment” ⟩
         (x + y = 20)[x ≔ x + 7]
       ≡⟨ Substitution ⟩
         x + 7 + y = 20
       ≡⟨ Fact `20 = 13 + 7` ⟩
         x + 7 + y = 13 + 7
       ≡⟨ “Cancellation of +” ⟩
         x + y = 13

This ‚Äòflipped around‚Äô notation is only superficial, and so it also applies to sequential commands, as follows. 

     Lemma “Demo”:  j + k = S
                   ⇒⁅ j  := j + 1⍮ k := k + 1 ⁆
                    j + k = S + 2
     Proof:
         j + k = S + 2
       ⁅ k := k + 1 ⁆⇐⟨ “Assignment” ⟩
         (j + k = S + 2)[k ≔ k + 1]
       ≡⟨ Substitution ⟩
          j + k + 1 = S + 2
       ⁅ j := j + 1 ⁆⇐⟨ “Assignment” ⟩
         (j + k + 1 = S + 2)[j ≔ j + 1]
       ≡⟨ Substitution — and sym & assoc of + ⟩
         j + k + 1 + 1 = S + 2
       ≡⟨ Fact `1 + 1 = 2` ⟩
         j + k + 2 = S + 2
       ≡⟨ “Cancellation of +” ⟩
         j + k = S
